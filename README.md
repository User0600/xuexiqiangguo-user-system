# xuexiqiangguo-user-system
Spring Boot用户管理系统 - 包含注册登录完整实现
## 📖 项目简介
一个完整的Spring Boot用户管理系统，实现了用户注册、登录、JWT认证等功能。适合初学者学习Spring Boot三层架构。
## 🚀 技术栈
- **后端框架**：Spring Boot 3.5.8
- **数据库**：MySQL
- **安全认证**：JWT Token
- **API文档**：Swagger/Knife4j
- **构建工具**：Maven
